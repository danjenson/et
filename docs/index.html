Edward Tufte Jekyll Theme

This is an Edward Tufte theme based on clayh53/tufte-jekyll. It is updated to be jekyll-remote-theme compatible and tested with ruby 3.1 and jekyll 3.9. You can learn how Jekyll sites work and how to customize them here.

To use this theme, add the following to your _config.yml:

remote_theme: danjenson/et
markdown: kramdown
plugins:
  - ...
  - jekyll-remote-theme
footer: # run `grep icon-` in project folder for icon names
  - link: https://twitter.com/edwardtufte
    icon: icon-twitter
  - link: https://github.com/danjenson/et
    icon: icon-github

Then, to run locally:

bundle add jekyll-remote-theme webrick
bundle install
bundle exec jekyll serve

You can also use latex with $$, e.g. $$f^2$$ becomes \(f^2\). To create a block, create a newline before the opening $$. You can see here the list of available mathjax commands.

Here are some quirks:

  1. Mathjax prefers subscripts inside tags, i.e. \mathbb{E_{\tau\sim P(\theta')}} renders $\mathbb{E_{\tau\sim P(\thetaâ€™)}}$, but \mathbb{E}_{\tau\sim P(\theta')} will not render correctly. Same for \vec{a_i} and \vec{a}_i.

Available liquid tags:

# Capitalizes words
<span class='newthought'>Example website: <a href='http://example.com'>example label</a></span> 
# Short quotation
<div class='epigraph'><blockquote><p> "How did you go bankrupt?" Two ways. Gradually, then suddenly.</p><footer>Ernest Hemingway, <cite> "The Sun Also Rises" </cite></footer></blockquote></div>
# Indexed note that appears in margin
<label for='sidenote-id' class='margin-toggle sidenote-number'></label><input type='checkbox' id='sidenote-id' class='margin-toggle'/><span class='sidenote'>This is a random sidenote </span>
# Non-indexed note that appears in margin
<label for='margin-note-id' class='margin-toggle'> &#8853;</label><input type='checkbox' id='margin-note-id' class='margin-toggle'/><span class='marginnote'>Random thought when drinking </span>
# Full-width image
<figure class='fullwidth'><img src='/assets/img/full-width.png'/><figcaption>A full-width image.</figcaption></figure>
# Main-column image
<figure><img src='http://example.com/image.jpg'/><figcaption class='maincolumn-figure'>Main column image.</figcaption></figure>
# Margin image
<label for='margin-figure-id' class='margin-toggle'>&#8853;</label><input type='checkbox' id='margin-figure-id' class='margin-toggle'/><span class='marginnote'><img class='fullwidth' src='/assets/img/side.png'/><br>This is the caption</span>
